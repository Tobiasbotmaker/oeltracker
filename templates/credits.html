Credits

Dette projekt bruger følgende open source-teknologier og ressourcer:

Backend

  • Flask - Web framework (MIT-licens)
  • Flask-SQLAlchemy - Database ORM (MIT-licens)
  • Flask-Migrate - Database migrations (MIT-licens)
  • Flask-WTF - CSRF-beskyttelse og formularhåndtering (BSD-licens)
  • Flask-Session - Server-side session management (MIT-licens)
  • Flask-Caching - Caching til forbedret ydeevne (BSD-licens)
  • Flask-Compress - Komprimering af HTTP-responser (MIT-licens)

Frontend

  • Bootstrap - Frontend framework til styling og layout (MIT-licens)

Andre ressourcer

  • HTTPS og sikkerhedsforanstaltninger implementeret med Flask's sikkerhedsfunktioner.
